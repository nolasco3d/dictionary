# Dictionary

Listagem de palavras, disponibilizadas através da dependência: __word-list-json__.

Demonstração do projeto neste [link](https://markdownlivepreview.com/).

## Tecnologias utilizadas

* Nuxt3
* Pinia
* VueUse
* Tailwind CSS

## Instruções de uso
```sh
# Instalando as dependências
$ yarn install 

# Rodando o projeto
$ yarn dev
```
